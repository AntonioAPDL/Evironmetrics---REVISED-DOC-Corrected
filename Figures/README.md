# Figures

This directory contains the article-facing figure files.

- `manuscript/`: the exact figure files used by `wileyNJD-APA.tex`
- `appendix_cutoff_panels/`: composite cutoff-specific setup/support panels kept for appendix use and review
- `forecast_context_by_cutoff/`: advisor-facing copies of the Figure 4 forecast-context view for all five cutoffs
- `multivariate_synthesis_by_cutoff/`: advisor-facing copies of the Figure 7 family for all five cutoffs
- `reference_synthesis_by_cutoff/`: advisor-facing copies of the Figure A2-style family for all five cutoffs
